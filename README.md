# OIBSIP - Java Development Internship Projects

This repository contains the projects completed during my Java Development Internship at Oasis Infobyte under the OIBSIP (Oasis Infobyte Internship Program).

The internship focused on strengthening core Java programming concepts, problem-solving skills, object-oriented programming, and console-based application development through real-world mini projects.

---

# Projects Included

## 1. Guess The Number Game

The “Guess The Number” project is a fun and interactive Java console application where the computer generates a random number within a specific range and the user attempts to guess the correct number.

The application continuously guides the user by informing whether the guessed number is higher or lower than the generated number until the correct answer is found.

### Features

* Random number generation using Java
* User input handling using Scanner class
* Hint system (Higher / Lower)
* Multiple guessing attempts
* Score calculation system
* Attempt limiting functionality
* Multiple rounds support
* Interactive console-based gameplay

### Concepts Used

* Loops
* Conditional Statements
* Random Class
* Scanner Class
* Methods and Functions
* Basic Game Logic

### Objective

The main objective of this project was to understand Java fundamentals, conditional logic, looping structures, and user interaction through console applications.

---

## 2. ATM Interface

The ATM Interface project is a console-based banking simulation application developed using Java. The project replicates the basic functionalities of an Automated Teller Machine (ATM) system.

The user is required to enter a valid User ID and PIN to access ATM services. After successful authentication, the user can perform various banking operations through a menu-driven interface.

### Functionalities Included

* User Login Authentication
* Transaction History
* Withdraw Money
* Deposit Money
* Transfer Money
* Balance Checking
* Exit/Quit Option

### Features

* Secure login system
* Menu-driven console interface
* Transaction recording system
* Balance management
* Input validation
* User-friendly interaction

### Concepts Used

* Object-Oriented Programming (OOP)
* Classes and Objects
* Methods
* Loops and Conditionals
* ArrayList / Data Storage
* Scanner Class
* Encapsulation

### Objective

The purpose of this project was to gain practical understanding of Java classes, real-world application flow, and implementation of banking operations using object-oriented programming concepts.
